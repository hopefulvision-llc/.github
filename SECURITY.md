# Security Policy

For security vulnerabilities, please follow these instructions to ensure a timely and private response.

## Reporting a Vulnerability

Please report security issues privately to: cosimos.portinari@gmail.com

Include:
- A short summary of the issue
- Steps to reproduce (or a test case)
- Affected components and versions
- Any relevant logs, screenshots, or proof-of-concept code
- Your contact information for follow-up

If you prefer, you may use GitHub's private security advisory feature to report vulnerabilities.

## Response Policy

- We will respond within 72 hours to acknowledge receipt.
- We will assess and provide an estimated timeline for remediation.
- We will coordinate disclosure with the reporter; public disclosure will be handled responsibly to reduce risk to users.
